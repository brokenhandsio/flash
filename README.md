#Installation

Update your `Package.swift` file.
```swift
.Package(url: "https://github.com/nodes-vapor/slugify", majorVersion: 1)
```

### main.swift
```
import Flash
```

And add provider
```
try drop.addProvider(FlashProvider.self)
